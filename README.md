## 操作方式
1. 鼠标右键控制角色移动
2. 按下 `Q` 键选择火球技能，点击鼠标左键即可发射
3. 按下 `F` 键选择闪现技能，点击鼠标左键即可释放闪现
4. 多人模式中 `Enter` 进入在线聊天室、 `Esc` 退出
